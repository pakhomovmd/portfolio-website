# Настройка отправки Email через Resend

## Что такое Resend?

Resend - это современный сервис для отправки email через API. Он предоставляет бесплатный план с 100 письмами в день, что идеально подходит для контактной формы на портфолио.

## Быстрая настройка (5 минут)

### Шаг 1: Регистрация на Resend

1. Перейдите на [resend.com](https://resend.com)
2. Нажмите "Sign Up" и зарегистрируйтесь (можно через GitHub)
3. Подтвердите email

### Шаг 2: Создание API ключа

1. После входа перейдите в раздел **API Keys**
2. Нажмите **Create API Key**
3. Дайте ключу имя (например, "Portfolio Contact Form")
4. Выберите разрешения: **Full Access** или **Sending Access**
5. Нажмите **Create**
6. **ВАЖНО:** Скопируйте ключ сразу (он показывается только один раз!)

### Шаг 3: Добавление ключа в проект

1. Создайте файл `.env` в корне проекта:
```bash
cp .env.example .env
```

2. Откройте `.env` и вставьте ваш API ключ:
```env
RESEND_API_KEY=re_ваш_ключ_здесь
CONTACT_EMAIL=pakhomovmichael@icloud.com
```

3. Сохраните файл

### Шаг 4: Перезапустите сервер

```bash
# Остановите dev-сервер (Ctrl+C)
# Запустите снова
npm run dev
```

## Проверка работы

1. Откройте сайт: [http://localhost:3000](http://localhost:3000)
2. Прокрутите до секции "Контакты"
3. Заполните форму и отправьте тестовое сообщение
4. Проверьте email (указанный в `CONTACT_EMAIL`)

## Альтернативы (если не хотите использовать Resend)

### Вариант 1: Без email (только логи)

Если не добавите `RESEND_API_KEY`, форма будет работать, но сообщения будут только логироваться в консоль сервера:

```bash
npm run dev
# Сообщения будут видны в терминале
```

### Вариант 2: Другие сервисы

Можете использовать:
- **SendGrid** - 100 писем/день бесплатно
- **Mailgun** - 5,000 писем/месяц бесплатно
- **Amazon SES** - требует AWS аккаунт

Для этого нужно будет изменить код в `app/api/contact/route.ts`

## Настройка домена (опционально)

Для production рекомендуется настроить свой домен:

1. В Resend перейдите в **Domains**
2. Нажмите **Add Domain**
3. Введите ваш домен (например, `pakhomovmd.com`)
4. Добавьте DNS записи (Resend покажет какие)
5. Дождитесь верификации (обычно 5-10 минут)

После этого можете отправлять с адреса типа `contact@pakhomovmd.com`

## Troubleshooting

### Ошибка: "Invalid API key"
- Проверьте, что скопировали ключ полностью
- Убедитесь, что в `.env` нет лишних пробелов
- Перезапустите dev-сервер

### Письма не приходят
- Проверьте папку "Спам"
- Убедитесь, что `CONTACT_EMAIL` указан правильно
- Проверьте логи в терминале на наличие ошибок

### Ошибка: "Rate limit exceeded"
- Вы превысили лимит 100 писем/день
- Подождите до следующего дня или обновите план
