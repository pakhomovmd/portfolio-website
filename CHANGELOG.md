- 🔍 SEO оптимизация

### Секции
- Hero - главный экран с анимированным фоном

- Skills - интерактивные карточки навыков с прогресс-барами

- Experience - таймлайн опыта работы
- Testimonials - карусель отзывов
- Contact - форма обратной связи с валидацией
- Footer - контакты и социальные сети

### Технологии
- Next.js 15+

## Формат

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

### Типы изменений
- **Добавлено** - для новых функций
- **Изменено** - для изменений в существующей функциональности
- **Устарело** - для функций, которые скоро будут удалены
- **Удалено** - для удаленных функций
- **Исправлено** - для исправления багов
- **Безопасность** - для исправлений уязвимостей
